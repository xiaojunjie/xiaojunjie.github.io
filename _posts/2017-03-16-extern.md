swap1出错，看下汇编  
{% highlight Python %}
//int data[]
movl	data, %eax
movl	data+4, %edx
movl	%eax, data+4
movl	%edx, data

//int * data
movl	data, %eax
movl	(%eax), %edx
movl	4(%eax), %ecx
movl	%edx, 4(%eax)
movl	%ecx, (%eax)
{% endhighlight %}
as we know, "movl data, %eax" 是有访存滴，"movl $data, %eax"是把data值送到eax。  
显然swap1的汇编是把data当int**  
实际在swap1输出的data为int，也就是1。
